Calculadora Básica

Funcionamiento de la calculadora

  • Pulsa 1 para sumar
  • Pulsa 2 para restar
  • Pulsa 3 para dividir
  • Pulsa 4 para multiplicar
  • Pulsa 5 para obtener el módulo
  • Pulsa 6 para salir

En función de la operación seleccionada, se realizará la operación correspondiente. Si pulsa la opción 6, se cerrará la ventana.
